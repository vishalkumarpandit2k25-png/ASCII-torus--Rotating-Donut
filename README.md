# ASCII Rotating Donut in C

This project displays a rotating 3D torus (donut) animation in the terminal
using ASCII characters. The animation is created using mathematical
projection and trigonometric functions in C.

## How to Compile and Run

### Linux / macOS
```bash
gcc donut.c -o donut -lm
./donut

## Program Behavior
The program runs continuously to display a smooth rotating animation.
Press Ctrl + C to stop the program.
